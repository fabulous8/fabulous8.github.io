Get the SolarEdge P400 Power Optimizer to handle some of the bigger wattage of today’s 72 or 96 cell solar panels. The P400 is rated for 400 watts of input from a 72 or 96 cell solar panel. A power optimizer is a DC/DC converter which is connected to the back of each PV module (or sometimes embedded by module manufacturers right off the pallet) which replaces the traditional solar junction box. Having MPPT on each module allows for flexible installation design with multiple orientations, tilts and module types in the same string. Be flexible and get the most out of your array! SolarEdge power optimizers increase energy output from PV systems by constantly tracking the maximum power point (MPPT) of each module individually. These power optimizers monitor the performance of each module and communicate performance data back to the SolarEdge monitoring portal so that you can keep a close eye on your array and detect any issues early. When working with SolarEdge Inverters, SolarEdge power optimizers automatically maintain a fixed string voltage, allowing installers even greater flexibility with longer strings and strings of different lengths in order to design optimal PV systems.
