Specially designed for use with ventilated floor tiles Cushioned T-handle grip allows for comfortable, easy lifting Designed to slightly raise panel so it may be manually removed Will not chip or damage surface covering Manufactured from heavy-duty strengthened steel

Regular panel lifters are unable to lift perforated panels because there is not enough solid area for a suction cup to grasp. This heavy-duty steel lifter has a hook that is specially designed for use with ventilated panels. The cushioned T-handle provides a firm grip which allows the user to quickly and efficiently lift panels without worrying about chipping the surface covering. The handle is designed to hang on any kind of holder, which can be purchased separately. Please note that this lifter is designed only to raise the panel out of the understructure so that it may be manually removed. Do not attempt to completely remove a panel using only a lifter.
