Here are some quick installation instructions…This material is not wood, it is flexible polyurethane resin and ships unfinished in a light tan color. Clean with acetone, mineral spirits or 91% rubbing alcohol (any solvent that evaporates quickly) If you are going to paint: Do not use pre-catalyzed primers/paints. No need to prime. BUT - Prime with Kilz if using an oil based paint. If you are going to stain: use gel based stains for best results (Minwax makes a great gel based stain) IF you need to fill nail holes: the best filler is epoxy (2 part syringe type) but Bondo for wood and “plastic wood” brand as well as Minwax wood fillers work well.Use a GEL type super glue for installation (Loctite PL premium adhesive works well, so does Liquid Nails)Use more glue and fewer nails. Use smallest gauge nail / brad possibleTry a 23ga pin nail gun. This is one of the best tools that has ever been invented for finish trim work. No dimples, no holes, no putty.Try to stay at least 1/4" away from the edges when nailing.
