Z-Wave Plus relay module adds On/Off commands to a single dry contact device Perfect for On/Off control of door and gate openers, water heaters, lights, and more Small device size allows for quick and easy installation in nearly any setting Control with toggle and pushbutton switches and Z-Wave On/Off commands Optional temperature sensor input enables smart automation scene triggers

Add wireless Z-Wave controls to a single electric device with the Qubino Flush 1D Relay. This universal relay module turns virtually any device or appliance with a large power supply (110-240 VAC/24 VDC) into a smart Z-Wave device to add into your Z-Wave network. This module is perfect for adding remote control to electric gate or door openers, heaters, water suppliers, or any other device that use a dry contact (110-240 VAC or 24 VDC). The Qubino Flush 1D Relay fits in behind a wall switch, between your selected device and switch, and requires a neutral wire for setup. If your selected device uses 24VDC power supply, it must be installed in a location between the power supply, the selected device, and a connected wall switch. Connecting a digital temperature or door sensor (sold separately) allows you to create automation scenes. Command fans or heaters to turn On/Off when a set temperature point is reached, or control garage doors or gates to Open/Close. The Qubino Flush 1D Relay also acts as a Z-Wave repeater to improve the range and stability of your Z-Wave network. Contents: Flush 1D Relay Module, Instruction Manual. NOTE: Remote commands using Z-Wave requires a Z-Wave network controller (sold separately). Manufacturer Part #: ZMNHND3. Warranty: 2 years.
