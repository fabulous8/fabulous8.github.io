These are woodturning project kits. A lathe is needed to make into finished pens. Blanks, bushings, drill bit, and all other turning accessories are not included. Includes (14) Premium Comfort Twist Ballpoint Pen Kits in seven different finishes. You will receive (2) of each of the following finishes: Gold (PKCFPEN), Brushed Satin (PKCFPENS), Black Enamel (PKCFPENB), Chrome (PKCFPENP), Gun Metal (PKCFPENBT), Copper (PKCFPENRG), and Titanium Nitride (PKCFPENTN). Features a 7mm dual tube design, twist mechanism, and a Cross style ballpoint black refill. Uses bushing set PKCFBUSH. Minimum blank size needed to turn is 5/8" x 5/8" x 4-1/2". Silicone comfort grip is included with each kit. You can make the pen with or without the comfort grip.
